이벤트 객체는 전역객체로써 모든 태그에 객체를 차조시킬 수 있다!

  • a태그와 button태그에 이벤트를 주로 참조시켜 클릭에 대한 반응을 기대할 수 있는 구조로 만드는게 일반적

  • a태그는 href 속성에 의해 자체 이동(동기화방식)이 구현되어서 별도의 이동속성을 막는 코드가 필요

  • 이벤트 객체를 부여하는 방식에는 크게 3가지 방식이 있다.

연습을 위해서 콘솔에 출력 / html문서(document영역)에 출력하는 방식

  • 화면출력 영역: document영역(객체)라고 한다.

  • html 태그는 html element, 속성 attribute, 일반 텍스트 textnode라고 함

  • html로 출력하기: innerHTML이라는 property를 사용
  • text로 출력하기: innertext라는 PROPERTY를 사용

  • html element를 id값으로 객체를 가져오기: getElementById('idName')

  • html element를 *css 선택자를 사용하요 가져오기: querySelector('#idName')
