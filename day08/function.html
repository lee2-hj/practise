함수(Function): 특정 기능을 수행하기 위한 코드 실행기 - 호출되어야만 작동

  • function 키워드를 사용하여 선언

  • 함수명: 기능을 함축하여 의미있는 함수명으로 작성

  • 함수명 뒤에( ) 파라미터를 사용하여 선언 할 수 있다.(없어도 무관)
    이때 파라미터값은 자유도가 높아 다양한 형태로 사용이 가능.

  • ( ) 다음 { } 기능 로직을 작성

  • return이라는 키워드를 사용하여 함수의 종료시점을 지정할 수 있음.

  • return은 함수의 실행 종료 역할과 동시에 결과값을 호출한 지점으로 반환 할 수 있다.

  • 함수 선언 방법은 크게 3가지로 구분.
