advanced for : 행상된 for문, 배열의 정렬

  • for of : for (let value of array){code...}: 배열을 하나씩 꺼내 출력
  • for in : for (let value in object){code...}: 객체를 하나씩 꺼내 출력
  • 배열의 정렬 : 정렬은 문자 기반으로 정렬이 됨.
  • array.sort() : 문자기준 순차 오름차순 정렬(ascending)
  • array.reverse() : 문자 기준 역정렬(descending) 내림차순
  • 정렬의 배열의 원소가 숫자인 경우 콜백함수를 이용하여 내부 연산기법을 활용한다.
  • array.sort(function(a,b){return a-b}): 숫자 오름차순
  • array.sort(function(a,b){return b - a}): 숫자 내림차순
