배열의 특정 기능을 할 수 있는 메서드

  • array.every(callback function(value,index,self)): callback함수의 리턴조건에 모든 배열의 값이 만족하면 true, 아니면 false
  • array.some(callback function(value,index,self)): 콜백함수의 리턴조건에 만족하는 배열값이 존재하면 true, 없으면 false
  • array.indexof("str value",[start index]): 배열의 값중 해당 문자열이 있으면 그 index값을 리턴
  • array.lastindexof("str value",[start index]): 배열의 값 중 해당 문자열이 있는 배열의 마지막 index를 리턴
  • array.find(callback function(value,index,delf)): 콜백함수의 리턴조건에 만족하는 첫번째 배열원소의 값을 리턴
  • array.findindex(callback function(value,index,delf)): 콜백함수의 리턴조건에 만족하는 첫번째 배열원소의 index를 리턴
  • array.from('순서가 있는 순환구조를 갖고있는 객체'): 배열로 변환함
  • array.keys(): 배열이 사용중인 key(식별자,index)를 검색해서 새로운 배열로 리턴 => 객체에는 적용이 안됨
  • array.entries() : 배열의 index:value의 구조의 값을 만들어 새로운 배열에 담아 리턴 => 객체에는 적용이 안됨.
  • array.includes("찾고 싶은 값"): 배열에서 찾는 값이 존재한다면 true
  • array.reduce(callback function(total,vlue,index,self),[init value]): 내부 연산의 누적값을 리턴
