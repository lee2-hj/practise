Map: key / value 형태로 데이터를 한 개의 배열로 만들어서 다시 배열로 저장하는 방식

  • 선언 및 초기화: new Map(), new Map(["key",value],["key",value],...)
  • 중첩 구조이기 때문에 key의 중복 확인이 필수 (중복 x)
  • key를 식별자로 이용하기 때문에 get("keyName"), set("keyName,value") 메서드 제공
  • 객체를 배열화 시킨 구조
  • 기본 key타입은 문자열이 원칙 키타입이 달리지면 관리하기 어려워짐 => 자바스크립트 객체들도 key는 문자로 사용.
