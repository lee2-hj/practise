DOM(Document Object Model): 문서 객체 모델
html tag를 element로 가져오는 메서드

  • 엘리먼트로 가져오는 메서드 byId, byClassName, byTagName
  • document.getElementById('idName') : 리턴단일 엘리먼츠
  • document.getElementsByClassName('className'): 리턴 복수 엘리먼트 : HTMLCollection 타입으로 저장
  • document.getElementsByTagName('tagName'): 리턴 복수 엘리먼트 : HTMLCollection 타입으로 저장
